# 手机打不开时的排查步骤

## 1. 先放行防火墙（最常见原因）

- 在项目里找到 **`开放端口-请右键以管理员运行.bat`**
- **右键** → 选 **“以管理员身份运行”**
- 按提示完成后再用手机访问

## 2. 确认手机和电脑在同一网络

- 手机用 **WiFi**，不要用流量
- 电脑也连 **同一个 WiFi**（不要一个 WiFi 一个网线不同网段）
- 若公司/学校网络有“设备隔离”，手机可能访问不到电脑，可换家里 WiFi 试

## 3. 确认电脑上的服务在跑

- 前端：在项目根目录运行 `npm run dev:client`，看到类似：
  - `Network: http://192.168.x.x:5176/`
- 后端：`npm run dev:server`，看到 `Server http://localhost:3000`

## 4. 用对地址和端口

- 在手机浏览器输入的是 **电脑的局域网 IP + 前端端口**，例如：
  - `http://192.168.110.216:5176/`
- 端口要以终端里 **Network** 那一行为准（可能是 5173、5175、5176 等）

## 5. 查电脑当前 IP（若不确定）

在电脑 PowerShell 里运行：

```powershell
ipconfig | findstr "IPv4"
```

用 **192.168.x.x** 或 **10.x.x.x** 那个地址，加上终端里显示的端口，例如：`http://192.168.110.216:5176/`

## 6. 仍不行时

- 暂时关闭 Windows 防火墙试一次（控制面板 → Windows Defender 防火墙 → 启用或关闭）
- 或用电脑开手机热点，让电脑连手机热点，再用手机访问电脑 IP（部分热点会禁止访问，不一定成功）
